Overview

== Definition: network::alias Creates an alias on an interface with a static IP address. There is an assumption that an aliased interface will never use DHCP. === Parameters: $ensure - required - up|down $ipaddress - required $netmask - required $gateway - optional $userctl - optional - defaults to false === Actions: Deploys the file /etc/sysconfig/network-scripts/ifcfg-$name. === Sample Usage: network::alias { 'eth0:1': ensure => 'up', ipaddress => '1.2.3.5', netmask => '255.255.255.0', } === Authors: Mike Arnold <mike@razorsedge.org> === Copyright: Copyright (C) 2011 Mike Arnold, unless otherwise noted.

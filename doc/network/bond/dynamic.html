== Definition: network::bond::dynamic Creates a bonded interface with static IP address and enables the bonding driver. bootp support is unknown for bonded interfaces. Thus no bootp bond support in this module. === Parameters: $ensure - required - up|down $mtu - optional $ethtool_opts - optional $bonding_opts - optional === Actions: Deploys the file /etc/sysconfig/network-scripts/ifcfg-$name. Updates /etc/modprobe.conf with bonding driver parameters. === Sample Usage: network::bond::dynamic { 'bond2': ensure => 'up', } === Authors: Mike Arnold <mike@razorsedge.org> === Copyright: Copyright (C) 2011 Mike Arnold, unless otherwise noted.
